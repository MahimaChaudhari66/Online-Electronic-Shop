# Online-Electronic-Shop
Online Electronic Shop is a full‑stack Java web application developed using Servlets, JSP, and JDBC. It is designed to simulate a dynamic e‑commerce platform where both customers and administrators can interact with the system. The project demonstrates practical implementation of database connectivity, session management, and CRUD operations in a Java web environment, making it a strong portfolio piece for fullstack development.

Admin Features
- Admin Login – Secure authentication for administrators.
- Add Product – Insert new electronic items into the catalog.
- Update Product – Modify product details such as name, price, or stock.
- View Products – Display all products stored in the database.
- Logout – End admin session securely.

 Customer Features
- Customer Login / Registration – Create an account or log in to shop.
- Browse Products – View available electronics dynamically from the database.
- Buy Product – Add items to cart and place orders.
- View Products – Check product details before purchase.
- Logout – End customer session securely.

 Technical Highlights
- Servlets handle business logic and request/response processing.
- JSP pages render dynamic content such as product lists, shopping cart, and order confirmations.
- JDBC enables database connectivity for CRUD operations (users, products, orders, cart).
- Session Management ensures secure login and cart persistence.
- Database Schema includes tables for users, products, orders, and cart items.
- UI/UX designed with HTML and CSS for clean, responsive layouts.
